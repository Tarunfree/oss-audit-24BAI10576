## About the Project
This project is an audit of an open-source software called Git. Git is a distributed version control system created by Linus Torvalds. It is one of the most widely used tools in software development. This project explains the origin of Git, its open-source license, how it works on Linux systems, and its role in the open-source ecosystem.

The project also includes 5 shell scripts that demonstrate Linux and shell scripting concepts such as variables, loops, conditions, file handling, and user input.

## Scripts Included

script1.sh – System Identity Report  
script2.sh – FOSS Package Inspector  
script3.sh – Disk and Permission Auditor  
script4.sh – Log File Analyzer  
script5.sh – Open Source Manifesto Generator
